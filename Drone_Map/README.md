## 드론대회 경기장 규칙

* 좌우 이동 경우의 수 (총 3가지)
   ``` 
   이동 없음
   좌 1200mm 이동
   우 1200mm 이동
   ```

* 상하 높이 경우의 수 (총 3가지)
   ```
   위   (상자 3개 높이) 
   중간 (상자 2개 높이)
   아래 (상자 1개 높이)
   ```

* 높이나 좌우의 경우, 최대 10cm의 오차가 있을 수 있습니다.

* 1단계 높이: `중간`
* 2단계 높이: `위` or `아래`
* 3단계 높이: 2단계의 `반대`
